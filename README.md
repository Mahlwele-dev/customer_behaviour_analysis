# 📊 Customer Behavior & Revenue Analysis  

## 🧠 Overview  
This project analyzes customer purchasing behavior and revenue patterns to identify key drivers of business performance.  

The goal is to understand **who drives revenue, what products perform best, and how customer segments differ**, enabling better data-driven decisions.

---

## 📁 Dataset  
The dataset contains transactional and customer-level data, including:  
- Customer ID  
- Transaction ID  
- Product Category  
- Purchase Amount  
- Review Rating  
- Age Group  
- Subscription Status  

---

## 🛠 Tools & Technologies  
- **Python (Pandas, NumPy)** → Data cleaning & preprocessing  
- **SQL (PostgreSQL)** → Querying and data analysis  
- **Power BI** → Dashboard and visualization  
- **Gamma / PowerPoint** → Reporting and presentation  

---

## 🔄 Project Workflow  

### 1. Data Cleaning (Python)  
- Handled missing values (e.g. filled ratings using category median)  
- Removed inconsistencies and ensured data quality  
- Prepared dataset for analysis  

---

### 2. Exploratory Data Analysis (EDA)  
- Analyzed customer purchasing patterns  
- Examined revenue distribution across categories  
- Identified key trends in customer behavior  

---

### 3. SQL Analysis (PostgreSQL)  
- Queried structured data to extract insights  
- Performed aggregations (revenue, customers, transactions)  
- Supported deeper analysis for reporting  

---

### 4. Dashboard Development (Power BI)  
Built an interactive dashboard to visualize:

- Total customers and average purchase value  
- Revenue by product category  
- Sales distribution across customer segments  
- Revenue and sales by age group  
- Subscription behavior

## 📊 Dashboard Insights  

- **Clothing category generates the highest revenue**, making it the primary driver of sales  
- **Young Adult customers contribute the most revenue**, indicating strong engagement in this segment  
- A large portion of customers are **non-subscribers**, suggesting an opportunity for conversion  
- Customer purchasing behavior varies across categories and demographics  

---

## 💡 Recommendations  

- Focus on **high-performing categories (e.g. Clothing)** to maximize revenue  
- Target **Young Adult customers** with personalized marketing strategies  
- Improve **subscription adoption** to increase customer retention  
- Use customer segmentation to drive targeted campaigns
